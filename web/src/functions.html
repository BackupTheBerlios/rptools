functions.js extends the capabilities of the dicetool by allowing you to write your own functions and have dicetool evaluate them in-line with the expression.

For example we provide one built in critical threat function called att(threat). att() rolls a d20 and compares it to threat. if d20==1, it prints 'miss' in the output expression column. if d20==20, it prints 'hit/threat' in the output expression column. if d20>=threat it prints 'threat' in the output expression column. This corresponds to the d20 critical threat rules.

You might use this like

att(15)+BAB+DEX;d6+1+STR
