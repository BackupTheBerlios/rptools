DiceTool

Introduction

We came up with dicetool while GMing. There were frequent multi-dice rolls and tabulation was slow sometimes. We looked at and tried many of the fine dice rolling apps out there, but they all lacked features we wanted (such as saving complicated expressions for re-rolling), and had features we didn't care for (such as cute pictures of the dice).

Features

  • user defined JavaScript functions
  • buttons have names to summarize the expressions
  • aggregating syntax for auto-summation
  • tabs for organizing by party, by encounter, however you wish
  • changes you make may be saved between sessions
  • 9 quick roll buttons modified by a quantity pulldown
  • variables on tool can be used in expressions or javascript functions
  • d20 style '1'='miss' and '20'='critical threat' rule highlighting
  • select (ctrl or shift-click) output rows for summation
  • platform-agnostic Java (either as WebStart or downloaded application)
  • Java 1.5 (WebStart supported)
