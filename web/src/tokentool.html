TokenTool

Introduction

With the popularity of maptool, finding images to use as tokens became a time consuming process. TokenTool makes creating elegant tokens easy.

Features

  • drag-and-drop any image onto the workspace
  • resize/align the image using your mouse
  • drag-and-drop the resultant token image directly onto maptool
  • Java 1.5 (WebStart supported)

Version

tokentool is currently released as 1.0M1. Roadmaps and release notes are in the Forums.
